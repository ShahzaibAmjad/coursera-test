Combination of Selectors in Style Sheets

  1. .{class} Any element with class1 will be affected.
  2. #{id1} Any element with id1 will be affected.
  3. {element1} Every element1 will be affected.
  4. {selector1}, {selector2} Same class will affect both selectors.
  5. {element1}.{class1} Every Element1 with class1 will be affected.
  6. {element1} > {element2} Element2 comes directly under Element1 will be affected. (Child Selector)
  7. {class1} > {element2} Element2 comes directly under Element with class1 will be affected.
  8. {element1} {element2} Every element2 comes indirectly under element1 will be affected. (Descendant Selector)
  9. {element1} {class1} Every element with class1 comes indirectly under element1 will be affected.
  10. .{class1}.{class2} Every element having both classes will be affected. To specify 2 or more classes write class = "class1 class2"
